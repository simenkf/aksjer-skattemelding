## Skjermingsfradrag calculator

As shown in the presentation, skjermingsfradrag involves a ton of manual computations. Fortunately, each year builds on the previous, so I've made a spreadsheet that will automatically perform all needed calculations: `skjermingsfradrag-kalkulator.xlsx`. Please read the start sheet of the excel document for instructions. Make sure to store your copy of the file in a location that will allow you to keep it for years to come, as you'll need updating it every single year for as long as you hold Accenture stocks (via ESPP or other non-Norwegian broker).

BTW: The excel sheet is Accenture/ESPP-agnostic. You can use it for any shares you have in other companies too. However, for any stock outside of ESPP, I highly recommend you use a Norwegian broker so that Skattemeldingen is automatically populated for you.
